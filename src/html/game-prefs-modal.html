About + Contact üìß

This project is a tribute to, and a modern twist on the original Armor Alley game.

The game has been ‚Äúremastered‚Äù based on the V1.1s1 Macintosh version graphics, and adapted to work on a range of modern screens and devices: desktops with a keyboard and mouse, tablets, and mobile phones using touch controls.

This began as a prototype in 2013, and has evolved to include nearly every aspect of the original game. It is a passion project, driven in part by kind words and stories of nostalgia from people who find it while looking up the classic PC or Mac versions.

Discord / Slack Integration üåêü™ù

Webhooks enable sharing of your game stats with others via Discord or Slack, popular gaming and real-time messaging apps.

Webhooks are effectively secret links you create on those services, allowing message posting to a specified channel. For Discord, see Preferences > Integrations > Webhooks. For Slack, see using incoming webhooks.

A note on security: Your browser posts directly to the webhook URL you provide. It is not logged or otherwise stored on the Armor Alley server.
